インストールと初期配置

1) インストール前の準備

  • 任意の場所に WEPP 用フォルダを作成します（推奨：Cドライブ直下）。
  • 例：エクスプローラ → PC → OS(C:) → wepp2024 というフォルダを作成

2) インストールの実行

  • ダウンロードした weppwin-2024-installer.exe を実行します。
  • 「WindowsによってPCが保護されました」と表示された場合は 「詳細情報」→「実行」を選択してください。
  • 表示されたら Next をクリック。
  • インストール先指定画面で Browse を押し、 先ほど作成した wepp2024 フォルダを選択 → Next。
  • デスクトップアイコン作成はチェックを入れて進めてOKです。
  • Install をクリックしてインストールを実行。
  • 完了画面で Launch WEPP にチェックを入れ、 Finish をクリック。

3) WEPPの動作チェック

  • 初回起動時の確認画面が表示されたら OK。
  • 「Use the default WEPP project」を選択し Start Task。
    「Don't show again」にチェックを入れておくと次回以降省略されます。
  • メイン画面が表示されます。
    画面下部の Run をクリックしてみましょう。
  • 右上に計算結果が表示されます（初期状態ではヤード・ポンド法）。
  • 画面上部の E/m アイコンをクリックするとメートル法表示に切り替わります。
    数値が表示されていれば、インストールは成功です。
